에피소드 7 · 구슬 두 개의 정체

46개인데 사실은 23쌍! 🧬

우리 몸 세포 안에 염색체가 46개 있다고 했지?

근데 이게 그냥 46개가 아니야.

1번~23번까지
각각 두 개씩 쌍으로 있어!

하나는 엄마한테서, 하나는 아빠한테서 받은 거야. 이게 바로 멘델 구슬 두 개의 정체야!

쌍이 된다는 게 무슨 말이야? 👫

예를 들어 1번 염색체를 보면—

엄마한테 받은 1번 염색체와 아빠한테 받은 1번 염색체가 같은 유전자들을 같은 위치에 갖고 있어. 같은 책의 엄마 버전, 아빠 버전이야!

멘델 구슬 두 개 = 이거였어! 💡

기억나? 멘델은 "구슬이 두 개씩"이라고 했어.

이제 정체가 밝혀졌어.

구슬 두 개
= 엄마 염색체의 유전자 하나
+ 아빠 염색체의 유전자 하나

같은 위치에 있는
두 가지 버전이야!

눈 색깔 유전자도 두 버전이 있어. 엄마 버전이 "파란 눈"을 시키고, 아빠 버전이 "갈색 눈"을 시킬 수 있어.

그럼 우성/열성이 뭐야? 🤼

두 버전이 있을 때 — 두 개가 다른 말을 하면 누구 말을 들어야 할까?

두 버전이 싸우면
이기는 쪽이 나타나 → 우성
지는 쪽은 숨어 → 열성

예)
갈색 눈 버전 vs 파란 눈 버전
→ 갈색 눈이 이겨서 갈색 눈으로 나타남
→ 파란 눈은 숨어서 안 보임
(하지만 자식한테 전달은 됨!)

멘델 실험이 다시 보여! 🔄

이제 멘델 실험을 유전자로 다시 설명할 수 있어.

자식한테 전달할 때 — 하나씩만! 🎲

자식을 만들 때 부모는 자기 두 버전 중 하나씩만 전달해. 무작위로!

Bb 부모가 전달할 수 있는 건:
→ B (보라색 버전) 또는
→ b (흰색 버전)

어느 쪽이 전달될지는
그때그때 달라 — 랜덤이야!

이게 바로 멘델이 발견한 "하나씩 무작위로 전달"의 실제 메커니즘이었어.

모건 이야기도 연결돼 🪰

ep3에서 아들은 X염색체가 1개뿐이라서 구슬이 하나뿐이라고 했지?

이제 정확히 이해돼.

딸: 1번~23번 염색체 모두 쌍으로
→ 유전자가 전부 두 버전씩

아들: 22번까지는 쌍으로,
23번(성염색체)만 X + Y
→ X에 있는 유전자는 버전이 하나뿐!
→ 그래서 흰눈 파리가 아들에게만 나왔어

감수분열 — 생식세포가 만들어지는 방법 🥚

부모가 자식한테 염색체를 전달할 때 — 46개를 통째로 줄 순 없잖아. 그러면 자식이 92개가 되잖아!

쌍 중에서 하나씩만 골라서 생식세포를 만들어. 그래서 정자와 난자는 각각 23개씩. 합치면 다시 46개!

이게 왜 중요하냐면 🌈

생식세포를 만들 때 쌍 중 어느 쪽을 고르는지가 랜덤이야.

23쌍에서 각각 하나씩 고르면
2²³ = 약 838만 가지 조합이 가능해!

여기에 교차(crossing over)까지 합치면
사실상 무한대에 가까운 조합.

그래서 형제자매도 다 다르고
세상에 똑같은 사람이 없는 거야!

정리해보면 🗺️

🧬 염색체 46개 = 23쌍
   엄마 버전 23개 + 아빠 버전 23개

🎱 멘델 구슬 두 개
   = 같은 위치의 두 버전 유전자

🤼 우성: 이기는 버전이 나타남
   열성: 지는 버전은 숨지만 전달됨

🎲 생식세포: 쌍에서 하나씩만 랜덤 선택
   → 자식마다 다른 조합

아직 안 푼 수수께끼가 있어 🤔

잠깐, 한 가지 이상한 게 있어.

ep1에서 배웠잖아.

37조 개 세포가
전부 똑같은 설계도(DNA)를 가지고 있어.

근데 왜 심장 세포는 심장이 되고
뇌 세포는 뇌가 되는 걸까?

설계도가 같으면 다 똑같이 만들어져야
하는 거 아닐까?

근데 말이야... 🤔

우성이 이기고 열성이 진다고 했잖아. 근데 진짜 궁금한 게 있어.

우성이 "더 좋은" 유전자라는 뜻일까?
항상 완벽하게 하나만 이기는 걸까?
혹시 둘 다 나타나는 경우도 있을까?

사실 우성·열성은
훨씬 더 복잡하고
흥미로운 이야기야!
